# Защищённая админка Sumka.tj

В Render добавь Environment Variables:

- `ADMIN_USER` — твой логин
- `ADMIN_PASSWORD` — сильный пароль
- `ADMIN_SECRET` — длинная случайная секретная строка

После деплоя админка открывается по `/admin`.

Публичный сайт может получать товары через `GET /api/products`, но добавление, редактирование и удаление товаров, а также просмотр/изменение/удаление заказов требуют входа администратора.
